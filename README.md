# ETHID-FINAL-SUBMISSION
EDITH OS

Even Dead, I Am The Hero

EDITH OS is a browser based operating system built using HTML, CSS and JavaScript. It recreates a desktop environment where users can open apps, manage windows and interact with a system just like a real OS. The goal was to combine design, logic and interactivity into one smooth and immersive experience.

⸻

Features

EDITH OS includes an interactive desktop with draggable and resizable windows along with a taskbar that tracks active applications and provides a start menu. It integrates an AI assistant powered by Gemini API and supports real time global chat using Firebase. The system also includes built in apps such as Notes, To Do, Calculator and Browser, along with local storage support to save user data seamlessly.

⸻

How it works

The system is controlled by a central OS object that manages windows, applications and user state. Apps are dynamically rendered inside movable window containers and behave like independent processes. Z index logic ensures proper window focus and layering, while data is handled using APIs and local storage to maintain responsiveness and persistence.

⸻

Tech stack

The frontend is built entirely using HTML, CSS and JavaScript without relying on frameworks. Firebase is used for authentication and real time database functionality, while the Gemini API powers the AI assistant. LocalStorage is used to store user data and settings directly in the browser.

⸻

What makes it different

This project goes beyond just UI design and focuses on simulating how an operating system actually behaves. Multiple applications can run simultaneously in a structured way, and the interaction design makes the experience feel closer to a real desktop environment rather than a traditional website.
